# 📊 Sales Dashboard | Power BI

A professional and interactive **Sales Dashboard** built using **Microsoft Power BI** to analyze sales performance, profit, orders, and customer behavior. The dashboard is powered by the **Sales_data.csv** dataset and provides interactive insights for business decision-making.

---

## 📌 Dashboard Preview

![Sales Dashboard](dashboard.png)

---

## 🚀 Features

- 📈 Total Sales Overview
- 💰 Total Profit Analysis
- 📦 Total Quantity Sold
- 🛒 Total Orders
- 👥 Customer Type Filter
- 🏷️ Category & Sub-Category Analysis
- 🌍 Region-wise Sales
- 📅 Order Date Filter
- 📊 Monthly Sales Trend
- 🗺️ State-wise Sales Analysis
- 🎯 Interactive Visualizations

---

## 📂 Dataset

This project uses **Sales_data.csv** as the primary dataset.

**Dataset Includes:**
- Order Date
- Customer Type
- Region
- Category
- Sub-Category
- Sales
- Profit
- Quantity
- State

## 📊 Dashboard Insights

- Office Supplies is the highest-selling category.
- Interactive filters allow analysis by Customer Type, Category, Region, and Date.
- Monthly Sales Trend helps identify sales patterns.
- State-wise analysis highlights top-performing locations.
- Sub-category breakdown provides detailed product performance.

## 🚀 How to Use

1. Clone this repository.

```bash
git clone https://github.com/babluyadav25/Sales-Dashboard.git
```

2. Open **Sales Dashboard.pbix** in Power BI Desktop.
3. If prompted, connect the data source to **Sales_data.csv**.
4. Refresh the data.
5. Explore the dashboard.
